#  College Helpdesk AI Chatbot

An intelligent AI-powered chatbot for college information and student support built with Python, Flask, and Natural Language Processing.

##  Features
-  Answers college-related queries (admissions, fees, placements, facilities, etc.)
-  Web interface with modern UI
-  Real-time conversation
-  Easy to deploy

## Tech Stack

- **Backend**: Python, Flask
- **NLP**: NLTK, scikit-learn, TF-IDF, Cosine Similarity
- **Frontend**: HTML, CSS, JavaScript
- **Deployment**: Ready for Heroku/Railway
